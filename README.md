# tags

For convenience, I tried to tag every session, so code can be reviewed per-video

# Installation and running

1. `git clone https://github.com/nikitaosyak/shootemupgamejs.git`
2. `cd shootemupgamejs`
3. `nmp install`
4. `gulp`

open http://localhost:8000/ in browser
